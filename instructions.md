1. Declare a variable called `madLib`.

2. The value stored in this variable should be a funny message, which makes use of the `adjective1`, `adjective2`, and `adverb` variables provided. (Hint: You may want to use string concatenation to craft your message!)

3. Since the values for each of these three variables will be captured by the prompt, we could have a different value stored in each of these every time we run our code. So try running your code or have a classmate run your code a few times and give different answers for your adjectives and adverbs each time!
